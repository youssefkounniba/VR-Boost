# Docs

Documentation assets for **vr boost agency**.

## Screenshots

Put screenshots of the app in [`screenshots/`](screenshots/). Suggested shots:

- `landing.png` — landing page (hero with the interactive 3D room)
- `dashboard.png` — dashboard
- `projects.png` — virtual visits table
- `project-detail.png` — visit detail (3D tour)
- `staging.png` — staging editor
- `hub.png` — Hub & Avatar flow

### Referencing them in the main README

```md
![Dashboard](docs/screenshots/dashboard.png)
```

Keep file names lowercase and hyphenated. PNG (UI) or JPG (photo-heavy) are both fine.
